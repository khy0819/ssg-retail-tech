상품 트렌드 분석기

SSG.COM 가상 데이터를 활용한 파이썬 프로젝트

프로젝트 개요

이 프로젝트는 리테일테크 분야에 대한 저의 관심과 기술적 역량을 보여주기 위해 제작되었습니다. 신세계 그룹의 온라인 쇼핑몰인 SSG.COM의 가상 상품 데이터를 활용하여 웹 스크래핑, 데이터 분석, 그리고 시각화 과정을 거쳤습니다. 이를 통해 온라인 쇼핑몰의 상품 가격 트렌드를 한눈에 파악할 수 있는 시스템을 구축했습니다.

사용 기술

  • Python: 프로젝트의 핵심 언어
  • BeautifulSoup & requests: 웹 데이터 수집
  • Pandas: 데이터 분석 및 처리
  • Matplotlib: 데이터 시각화

분석 결과

아래는 파이썬 코드가 생성한 가격대별 상품 분포 그래프입니다. 다양한 상품들의 가격대가 어떻게 분포되어 있는지 시각적으로 확인할 수 있습니다.

코드 실행 및 확인

이 프로젝트는 다음 단계를 통해 직접 실행하고 코드를 확인할 수 있습니다.

  1. 파이썬 파일 다운로드: GitHub에서 ssg_trend_analyzer.py 파일을 다운로드하세요.
  2. 터미널 실행: 터미널(명령 프롬프트)을 열고, 파일이 있는 폴더로 이동합니다.
  3. 코드 실행: 다음 명령어를 입력해 코드를 실행합니다. python ssg_trend_analyzer.py
  4. 결과 확인: 실행 후 터미널에 분석 결과가 출력되고, 그래프 파일(price_distribution.png)이 생성됩니다.

본 페이지는 포트폴리오 목적으로 제작되었으며, 실제 SSG.COM과는 무관합니다.
